## необходимо для запуска (минимально)

- **Python 3.10+**
- **Node.js 18+** и **npm**

### далее

> **важно**  
> **все команды должны вызываться из корня проекта (там где указано)**  

теперь нужно создать + активировать виртуальное окружение и установить зависимости (вызываем из корня)
```bash
python -m venv .venv
& .venv\Scripts\activate
pip install -r requirements.txt
```

теперь создаем в папке backend файл .env и вписываем туда следующее:
```bash
DATABASE_URL=sqlite+aiosqlite:///quiz.db
SECRET_KEY=asdasdasdasdasd
ENV=dev
```

#### устанавливаем зависимости для фронтенда (также из корня)
```bash
cd frontend
npm install
```

в папке frontend также создаем файл .env и вписываем:
```bash
VITE_API_URL=http://localhost:8000
```

### запуск веб апп

нужно запускать два сервера, один это фронтенд, второй бекенд

#### в первом терминале где мы устанавливали зависимости пайтона
```bash
uvicorn main:app
```

сразу бекенд стартует на юрл: **http://localhost:8000** (можно уже выполнять api запрос на эндпоинт)
- документация по api которую предоставляет fastapi по дефолту: **http://localhost:8000/docs** (в продакшене указывать ENV=prod в env файле бекенда, чтобы docs и прочие дев-юрл были недоступны)

#### во втором терминале где для node
```bash
cd frontend
npm run dev
```

фронтенд стартует на юрл: **http://localhost:3000** (либо смотрите какой юрл напишет в консоли)

все готово)
