## необходимо для запуска (минимально)

- **Python 3.10+**
- **Node.js 18+** и **npm**

### далее

> **важно**  
> **все команды должны вызываться из корня проекта (там где указано)**  

#### backend (вызывать из корня)
```bash
cd backend
pip install -r requirements.txt
```

#### фронтенд (также из корня)
```bash
cd frontend
npm install
```

далее можно закрывать эти терминалы

### запуск веб апп

нужно запускать два сервера, один это фронтенд, второй бекенд

#### в первом терминале
```bash
cd backend
python -m uvicorn main:app --reload
```

сразу бекенд стартует на юрл: **http://localhost:8000** (можно уже выполнять api запрос на эндпоинт)
- документация по api которую предоставляет fastapi по дефолту: **http://localhost:8000/docs**

#### во втором терминале
```bash
cd frontend
npm run dev
```

фронтенд стартует на юрл: **http://localhost:3000**

все готово, можно стартовать
